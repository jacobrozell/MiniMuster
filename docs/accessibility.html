Accessibility — MiniMuster (iOS)

Last updated: June 15, 2026
App: MiniMuster
Bundle ID: com.jacobrozell.minimuster

Overview

MiniMuster is designed to work with built-in iPhone and iPad accessibility features. You can browse armies and units, manage paints, change settings, import and export data, and use the home-screen widget with the supports listed below.

Supported features

VoiceOver

Lists, buttons, tabs, and swipe actions include accessibility labels and hints. Examples:

  • Army and unit rows announce name, faction, progress, and painting state.
  • Swipe actions on units are labeled (for example, advance painting state, duplicate, delete).
  • Collection overview and stat headers announce progress percentages.
  • Paint rows and grid items announce name, type, brand, and quantity.
  • Status banners are announced when they appear.

Turn on: Settings → Accessibility → VoiceOver (or ask Siri to turn VoiceOver on).

Larger Text

The app uses Dynamic Type and adapts layout at larger text sizes. At accessibility sizes, list rows may stack content (for example, unit name above the state chip), stat grids move to a single column, and iPad split-view columns widen to fit text.

Turn on: Settings → Accessibility → Display & Text Size → Larger Text (adjust the slider or enable Larger Accessibility Sizes).

Dark Interface

MiniMuster supports light and dark appearance. You can follow the system setting or choose a theme in the app.

In the app: Settings → Theme → Dark, Light, or System.

System-wide: Settings → Display & Brightness → Appearance.

Reduced Motion

When Reduce Motion is enabled, decorative animations are minimized or removed — including splash and onboarding transitions, progress ring animation, and banner slide-in effects. Core functionality is unchanged.

Turn on: Settings → Accessibility → Motion → Reduce Motion.

Common tasks

With the features above enabled, you should be able to complete everyday tasks in MiniMuster, including:

  • First launch and onboarding
  • Browsing and searching your collection
  • Opening army and unit details, editing notes, and advancing painting states
  • Managing paint inventory
  • Changing theme and other preferences in Settings
  • Importing, exporting, and backing up your data
